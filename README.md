# Project Summary

The TS Academy Capstone Project is a collaborative React-based web application developed as the culmination of three months of frontend development training at TS Academy. The primary purpose of this project was to enable team members to work collaboratively while applying the knowledge and skills acquired throughout the program, including HTML, CSS, JavaScript, React, Fetch API, responsive web design, Git, and GitHub workflows.

The project involved recreating a provided design mockup as a fully functional web application using a component-based architecture. The application integrates dynamic planetary data from an external API, displays planet information such as names, images, and distances from the sun, includes form submission with validation, responsive layouts, interactive navigation, and multimedia content.

Through collaborative development, team members worked on separate branches, submitted contributions through pull requests, reviewed each other's code, and followed industry-standard Git workflows. This project provided practical experience in teamwork, version control, problem-solving, documentation, and deploying a modern web application.

### Instructions on how to run the Project Locally on your machine

### Prerequisites

Before running the project, ensure the following are installed:

- Node.js (v18 or later recommended)
- npm (comes with Node.js)
- Git
- Visual Studio Code (recommended)
- Then have a working GitHub Account for later storage of cold if the need arises

### Step 1: Clone the Repository

```bash
git clone https://github.com/ValentinePraise/TS-Academy-Capstone-Project27
```

### Step 2: Navigate to the Project Folder

After cloning, enter the repository folder:

```bash
cd TS-Academy-Capstone-Project27
```

Inside the repository, the React application is located in the `my-react-project27` directory.

Navigate into it:

```bash
cd my-react-project27
```

### Step 3: Install Dependencies

```bash
npm install
```

This will install all required packages and dependencies for the project.

### Step 4: Start the Development Server

```bash
npm run dev
```

### Step 5: Open the Application

After running the command above, Vite will provide a local development URL similar to:

```text
http://localhost:5173
```

Open the URL in your browser to view the application.

### Build for Production

To generate an optimized production build:

```bash
npm run build
```

### Preview the Production Build

To preview the production build locally:

```bash
npm run preview
```
